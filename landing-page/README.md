# Agency-Quality Responsive Landing Page

A premium, fully responsive landing page built with **React (Vite)** and **Tailwind CSS**. This project translates a written brief into a professional, agency-standard web interface with a focus on clean hierarchy, performance, and modern aesthetics.

## 🚀 Features

- **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop screen sizes.
- **Modern Tech Stack**: React 19, Vite 8, and Tailwind CSS 4.
- **Agency-Grade Design**: Dark mode by default with subtle gradients, sharp typography, and interactive components.
- **Performance Focused**: Minimal dependencies and optimized semantic HTML for high Google PageSpeed scores.

## 📂 Project Structure

- `src/components`: Modular React components (Header, Hero, Services, Contact, Footer).
- `src/data`: Centralized site content for easy maintenance.
- `src/styles`: Tailwind CSS configuration and global styles.

## 🛠️ Setup & Development

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Locally**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
